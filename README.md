# Computer Graphics — Algorithm Implementations

A complete Jupyter notebook collection implementing fundamental computer graphics algorithms, from rasterisation to 3D projections and L-systems.

## Notebooks

| Notebook | Topic |
|----------|-------|
| `Line Drawing Algorithm.ipynb` | Basic line rasterisation |
| `Bresenham's Line Algorithm.ipynb` | Optimised integer-based line drawing |
| `Midpoint Circle Algorithm.ipynb` | Efficient circle rasterisation |
| `Liang Barsky Line Clipping Algorithm.ipynb` | Parametric line clipping against a rectangular viewport |
| `Interpolation Techniques.ipynb` | Linear, bilinear, and bicubic interpolation |
| `Orthographic Projection and Perspective Projection.ipynb` | 3D to 2D projection transforms |
| `Translational, Rotational, Shearing, Scaling, Reflection for 2D & 3D Images.ipynb` | Full affine transformation suite |
| `Face Morphing.ipynb` | Image warping and cross-dissolve morphing |
| `Lindenmayer System.ipynb` | L-system grammar for procedural plant/fractal generation |
